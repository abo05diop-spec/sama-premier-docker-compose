# Mon Premier Projet Docker Compose / My First Docker Compose Project

## 🇫🇷 Français
Ce projet permet de déployer rapidement un environnement web complet comprenant un serveur *Nginx* et une base de données *MySQL* en utilisant Docker Compose. C'est une base solide pour apprendre l'administration système et l'orchestration de conteneurs.

### Architecture :
* *Nginx* : Serveur Web / Reverse Proxy
* *MySQL* : Base de données
* *Docker Compose* : Orchestration des conteneurs

---

## 🇺🇸 English
This project demonstrates how to quickly deploy a complete web environment featuring an *Nginx* web server and a *MySQL* database using Docker Compose. It serves as a foundation for learning system administration and container orchestration.

### Tech Stack:
* *Nginx*: Web Server / Reverse Proxy
* *MySQL*: Database
